###  图标说明
1. 桌面和安装图标，修改里面的3个图片（图片名称不要修改）。
```
# 文件路径
electron-egg\build\icons

# 图1：256x256.png
# 分辨率：256 x 256
# 宽度：256像素
# 高度：256像素
# 位深度：32

# 图2：512x512.png
# 分辨率：512 x 512
# 宽度：512像素
# 高度：512像素
# 位深度：32

# 图3：icon.ico
# 分辨率：256 x 256
# 宽度：256像素
# 高度：256像素
# 位深度：32
```

2. 托盘图标
```
# 文件路径
electron-egg\public\images\tray_logo.png
分辨率：16 x 16
宽度：16像素
高度：16像素
位深度：32
```
