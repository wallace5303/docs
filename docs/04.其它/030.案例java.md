`electron`技术本身提供 `node.js`、`c/c++`(.node扩展) 语言支持。但为了拓展不同技术栈的开发者都能使用electron技术，
框架提供了`cross模块`，来创建其它语言的服务（子）进程。同时提供统一的API来管理这些进程。

以下案例是社区用户的 `商业`或`开源`项目。
