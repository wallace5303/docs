> ee-core：v2.0.3

###  介绍
框架EE模块，提供app对象。

注：子进程中无法使用所有涉及electron的api
###  使用方法
```javascript
const EE = require('ee-core/ee');
```
###  API列表
###  CoreApp
等于this.app对象。详情见：[CoreApp](/pages/4cc0ef/)
```javascript
const { CoreApp } = EE;
```
