###  介绍
> ee-core: v2.2.0
插件模块，可以调用框架内置或者开发中自己编写的插件。

### 使用方法
```javascript
const Addon = require('ee-core/addon');

# 内置window插件
const win = Addon.get('window').create('window-name', {title: 'demo'});

# demo-托盘插件
Addon.get('tray').create();
```

### API列表
### get()
获取实例化后的插件对象。
```javascript
Addon.get('window');
```
