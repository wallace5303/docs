> ee-core：v2.0.3

### 介绍
工具库helper模块。

### 使用方法
```javascript
const UtilsHelper = require('ee-core/utils/helper');
```

### API列表
### fnDebounce()

- fn [Function] - 回调函数
- delayTime [Time] - 延迟时间(ms)
- isImediate [Boolean] - 是否需要立即调用
- args [type] - 回调函数传入参数

去除抖动器。

### getRandomString()
随机10位字符串

### mkdir(filepath)

- filepath [String] - 文件路径

创建文件夹

### chmodPath(path, mode)

- path [String] - 文件路径
- mode [String] - 权限；如755、777

修改文件权限

### compareVersion(v1, v2)

- v1 [String] - 版本号
- v2 [String] - 版本号

版本号比较

### stringify(obj, ignore = [])
> ee-core：v2.1.0

- obj [String] - 对象
- ignore [Array] - 忽略对象中的属性

序列化对象为JSON字符串

### validValue(value)
> ee-core：v2.1.0

- value [String]

给定的参数是否为有效值，即：非 undefined / null / ''
