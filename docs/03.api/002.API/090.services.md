###  介绍
> ee-core：v2.2.0
服务层模块

### 使用方法
```javascript
const Services = require('ee-core/services');

# 获取service/storage.js 实例化对象
Services.get('storage');
```

### API列表
### get()
获取实例化后的service对象。
```javascript
Services.get('framework').doCreatePool(num, event);
```
