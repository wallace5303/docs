    electron/window

    ee-core：v2.2.0

    # 介绍

    框架封装的electron window功能，将在此模块提供API。

    注：子进程中无法使用所有涉及electron的api

    # 使用方法

    const CoreElectronWindow = require('ee-core/electron/window');

    # API列表

    # getMainWindow()

    获取已经创建的主窗口对象。

    # createMainWindow()

    创建electron主窗口，ee-core使用，开发者忽略。

    # restoreMainWindow()

    还原主窗口。
