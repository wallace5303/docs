    electron

    ee-core：v2.0.3

    # 介绍

    框架封装的electron功能，将在此模块提供API。

    注：子进程中无法使用所有涉及electron的api

    # 使用方法

    const Electron = require('ee-core/electron');

    # API列表

    # mainWindow

    返回主窗口对象。包含BrowserWindow的所有实例api

    Electron.mainWindow

    # extra

    返回 Object 。其中closeWindow属性，用来控制窗口退出。
