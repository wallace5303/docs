    开始

    ee-core: v2.8.0

    java 拥有庞大的生态、市场、开发者，该文档会介绍如何把java项目集成到桌面端。

    # 准备

    1. 下载项目
    # gitee
git clone https://gitee.com/dromara/electron-egg.git

# github
git clone https://github.com/dromara/electron-egg.git
    1. 切换 demo 分支
    git checkout -b demo-java remotes/origin/demo
    1. 安装 前端、electron、依赖
    # 根目录安装electron依赖
npm i

# 进入前端目录安装依赖
cd ./frontend 
npm i

    # 下载demo所需资源

    1. jre 和 jar
    • 地址：java-res.zip (opens new window)
    • 请根据你的操作系统，选择正确的 jre 并解压
    1. 将jre解压到 build/extraResources 目录

    备注：官方jre下载地址：Oracle java (opens new window)

    # 运行

    npm run start

    查看demo的cross菜单，Java服务。
